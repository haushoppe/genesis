About the Ordinal Cubes

Intro for visitors
The world's smallest onchain gallery, visualized as a 3D cube. Permissionless. Immutable. Forever existing. Choose six images that are already on the 🟧 Bitcoin blockchain and claim them with your own cube!

Detailed Description
Ordinal Cubes is a permissionless art project, anyone can create an onchain gallery, displaying existing images from the Bitcoin blockchain. The images are visualised as a 3D cube, which looks cool and holds the viewer's attention for a longer time. The created ordinals use recursive inscriptions heavily, so the cost of an inscription for each cube is very affordable.

Xverse Integration Partnership

Testimonial (by Johannes)
I didn't want to create an art collection where all works are already known at the beginning. Therefore, it was my goal that all parameters for the mint can be entered by the collector - inspired by fx(params). But the effort involved in running a dedicated instance of ord is high, and I wanted to invest my energy in an engaging website instead. For the project's initial launch, I utilized a hosted inscription service, which is the de-facto standard for most "lazy mints".

However, this approach requires trust in a third party. The inscription services create an order for you, and after you pay for the order, they hopefully start the inscription. In line with the Bitcoin ethos of self-custody and trustlessness, the optimal approach is for my collectors to create the transaction on their own, using their own device and private key. This is where Xverse comes in. With Xverse, you can pay directly to a commit address, avoiding extra fees, unnecessary wait times, and any intermediary!

How the Inscription API is used
The website detects if Xverse is installed. If Xverse is not installed, a message appears encouraging you to install Xverse. If Xverse is installed, there's a login button in the top right corner of the page, as well as inline text suggesting you log in with Xverse.
The login popup asks you to share your address to receive ordinal inscriptions. This address is then pre-filled in the minting form. It's now your turn to shuffle through the cube suggestions via "Craft another cube" until a pleasing suggestion appears.
Then, click the "Mint my cube" button. A popup from the Xverse wallet will appear, showing you the inscription that will be created for you. If you confirm this dialog, a transaction is created that will inscribe your cube as soon as a Bitcoin miner picks up the transaction.
